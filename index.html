Next backend layer to make this production-ready

Image backend

Use Sharp or ImageMagick for high-quality resizing, compression, and reliable format conversion.

PDF backend

Use qpdf, pdfcpu, or Ghostscript for page operations. Use a dedicated PDF editor pipeline for real text edits.

Document backend

Use LibreOffice headless for DOCX ↔ PDF conversion. Do not fake this in-browser because it breaks formatting.

EPS support

Use Ghostscript for EPS ↔ PDF and EPS → PNG conversion. Browser-only support here is weak and unreliable.
